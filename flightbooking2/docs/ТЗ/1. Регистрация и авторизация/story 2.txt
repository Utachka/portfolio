Заголовок и описание истории (Story):
    Как зарегистрированный пользователь, я хочу авторизоваться в системе, чтобы получить доступ к своему аккаунту и иметь возможность выйти из системы.


Описание Use Case:
    Акторы:
        Пользователь (новый пользователь системы)
        Зарегистрированный пользователь (пользователь у которого есть учетная запись в системе)
        Авторизованный пользователь (зарегистрированный пользователь, который прошел процесс авторизации)

Основной поток:
    1. Пользователь открывает главную страницу веб-приложение.
    2. Система проводит проверку, авторизован ли пользователь, если не авторизован отображает главную страницу приложения, на которое есть кнопка входа в систему.
    3. Пользователь нажимает кнопку войти.
    4. Система отображает страницу входа в систему.
    5. Пользователь вводит учетные данные и нажимает кнопку войти
    6. Система проверяет наличие такого пользователя в системе
    7. Система авторизует пользователя
    8. Система отображает главную страницу приложения, где отображен логин авторизованного пользователя и кнопка выхода из системы
    9. Ползователь нажимает кнопку выход
    10. Система удаляет сессию пользователя
    11. Система отображает главную страницу приложения, на которой есть кнопка входа в систему.


Альтернативные потоки:
    Шаг 2. Если пользователь авторизован, то система отображает главную страницу приложения, имя авторизованного пользователя и кнопку выхода из системы
    Шаг 6. Пользователь в системе не найден, система возвращает сообщение об ошибке


Описание REST API:
     paths:
  /login/:
    post:
      summary: Вход пользователя
      description: Позволяет пользователю выполнить вход в систему
      requestBody:
        required: true
        content:
          application/json:
            schema:
              type: object
              properties:
                username:
                  type: string
                  description: Имя пользователя
                password:
                  type: string
                  description: Пароль пользователя
              required:
                - username
                - password
      responses:
        '200':
          description: Успешный вход в систему
        '400':
          description: Ошибка валидации данных
        '401':
          description: Неверные учетные данные пользователя
      examples:
        'application/json':
          - name: Пример запроса
            value: |
              {
                "username": "john_doe",
                "password": "secret"
              }
          - name: Пример ответа
            value: |
              {
                "message": "Успешный вход в систему"
              }
  /logout/:
    post:
      summary: Выход пользователя
      description: Позволяет пользователю выйти из системы
      responses:
        '200':
          description: Успешный выход из системы
      examples:
        'application/json':
          - name: Пример ответа
            value: |
              {
                "message": "Успешный выход из системы"
              }
